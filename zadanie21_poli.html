Co znaczy skrót ID? Identyfikator
Jaki jest celstosowania ID? Unikalna identyfikacja obiektów; Ułatwienie zarządzania danymi; Efektywność przetwarzania.
Jakie jest znaczenie znaku # Definiowanie identyfikatora
Jak następuje wywołanie ID (wraz z wytłumaczeniem)? Wywołanie ID to uzyskanie unikalnego identyfikatora obiektu w systemie.
Uwagi na temat stosowania ID (trzy) konieczne wyliczenia. Unikalność(ID musi być unikalne); Bezpieczeństwo(Unikaj ujawniania wrażliwych danych); Stabilność(Unikaj zmian ID bez potrzeby).
